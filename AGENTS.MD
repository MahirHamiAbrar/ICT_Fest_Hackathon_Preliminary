## get started

1. install the dependencies

```bash
pip install -r requirements.txt
```

2. initiate .venv

```bash
python -m venv .venv
```

3. activate the .venv

```bash
source .venv/bin/activate
```

4. run the tests

```bash
pytest
```

5. run individual test

```bash
pytest tests/test_rate_limit.py
```
